# Doqment PDF Viewer

Doqment is a modern, lightweight, and eye-friendly PDF viewer for the Linux desktop. It brings the power of Mozilla's HTML5 PDF.js engine and integrates it seamlessly with GTK to provide a native experience.

## Features
- **Reader Mode (Dark Mode):** Protect your eyes with built-in color themes and dark mode, perfect for reading long documents at night.
- **Native GTK Interface:** Beautiful, clean HeaderBar that fits perfectly into the GNOME desktop environment.
- **Standalone App:** Runs completely independently of your web browser.
- **Secure:** Sandboxed via Flatpak with strict permissions.
